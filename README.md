## 📖 About This Repository
This repository contains my source code, problem sets, and the final project developed to earn the **CS50's Introduction to Programming with Python (CS50P)** certificate, offered by Harvard University through edX. 

The programs within this repository showcase a clear learning progression, starting from fundamental programming concepts and advancing towards intermediate-level software development. This journey highlights my ability to write clean, efficient, and Pythonic code to solve logical and computational problems.

## 🧠 Core Topics & Skills Mastered
The projects in this repository cover a wide range of critical programming paradigms and tools, including:

* **Fundamentals:** Variables, Data Types, Conditionals, and Loops.
* **Error Handling:** Exceptions, debugging, and writing robust code that anticipates user errors.
* **Libraries & APIs:** Importing built-in modules, installing third-party packages, and fetching data from external APIs.
* **Unit Testing:** Using `pytest` to write test cases and ensure code reliability and quality.
* **File I/O:** Reading, writing, and manipulating data across different file formats (TXT, CSV, JSON).
* **Regular Expressions (RegEx):** Pattern matching for data validation, parsing, and cleaning (Crucial for data analysis).
* **Object-Oriented Programming (OOP):** Classes, objects, methods, inheritance, and encapsulation for building scalable software architecture.

## 📂 Repository Structure
The repository is organized by the course's logical progression (Weeks/Modules). Each folder contains specific problem sets focusing on a new concept:

* `Functions_Variables/`: Basic syntax and mathematical operations.
* `Conditionals_Loops/`: Flow control and iteration.
* `Exceptions/`: Handling `ValueError`, `TypeError`, and custom exceptions.
* `Libraries/`: Command-line arguments, API requests, and package management.
* `Unit_Tests/`: Implementing `pytest` for quality assurance.
* `File_IO/`: Data extraction and manipulation (CSV/JSON processing).
* `Regular_Expressions/`: Data cleaning and string validation.
* `Object_Oriented_Programming/`: Advanced class structures and methods.
