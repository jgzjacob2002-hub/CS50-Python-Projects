# SWIMMING TIMES APP
### Video Demo:  <https://youtu.be/13rzDg-0Yng>
## Description:

<p align="justify">
This CS50 project is about swimming times, and it is similar to applications where you can see any type of information about athletes, the classification of any type of event, or even statistics where you can compare any type of information. My code focuses on swimming times, but it can even be used in other types of sports where time is especially important, so it is particularly useful in the world of sports.
<p align="justify">
My code is divided into three aspects, each with a different function. The first function is to sort a CSV, so this code obtains a list of random positions of the athletes and, taking into account the swimmer's time, they can obtain a better position in the ranking, so that the ranking order goes from the best time to the worst. This is useful because in sports where time is important, rankings are indispensable for seeing the athlete's position. In fact, it is not only useful for obtaining the position, but also for seeing who wins the competition or even which athlete qualifies for another important competition.
<p align="justify">
Secondly, my code can obtain certain statistics, such as the average time, the best time in the competition, the worst time, and the median. This is extremely useful, as you can compare these statistics and use them for whatever you want, to obtain a list of the best times in any event or the time limits in the competition. It depends on the person using the code, but in the world of sports, these statistical selections are very practical.
<p align="justify">
Thirdly, I added a search engine, which is also especially useful in the world of sports. You can search for any swimmer on this list or in the competition and see the athlete's time, name, and country. This application is useful when you can only see information about one swimmer.
<p align="justify">
These are the main functions of my code, but as an extra feature, I have added the possibility of viewing this application at any type of swimming event or other sporting events. Another extra feature is loops, which means that when you enter incorrect information when required, the code does not exit immediately but remains in that “menu.” So do not worry about making a mistake and thinking you must re-enter the information Finally, as a little extra, there are two menus from which you can exit the program.
</p>

## How it works
<p align="justify">
1.	The code will ask you to write a number, this number is between one and seven, and these are just the freestyle events in swimming, you can select between **50-, 100-, 200-, 400-, 800- and 1500-meters freestyle**. Also, you have the option of exit the execution. If you make a mistake, writing a letter or if you pass the amount of numbers, you will get a sentence, where it will require you to write the correct number.
<p align="justify">
2.	If you write a number between one and six, the code will ask you again to insert another number, this time between one and four.

    - First option: If you write one, you will see a ranking, this ranking depends on the CSV and the event that you select, so the ranking works according to the times. If the swimmer gets the best time, he will be at the top of the ranking, and if he gets the worst, he will be at the bottom of the ranking. (The ranking is tabulate)

    - Second option: If you write two, you will see four statistics; best time, worst time, average and median of the event.

    - Third option: If you write three, the code will ask you for a name that you want to search. Obviously, if you print the incorrect name, the code will ask you for the correct one. When you write the correct name, on the screen you will see a sentence, where you will see the name, the time, and the country of the athlete.

    - Fourth option: If you write four, you will exit the execution.
